Bienvenidos

Esta es una página web para la clase de Fundamentos de la Ingeniería Informática del primer curso de Ingeniería Informática.

Su objetivo es documentar y mostrar los conocimientos adquiridos en HTML y CSS, así como mi progreso académico dentro del grado.
