Sofía Pérez Puerta

Esta es una pagita web para la clase de fundamneto de la ingeniera informatica del primer curso de Ingenieria Informatica.

Su objetivo es documentar y mostrar los conocimientos adquiridos en HTML y CSS, así como mi progreso académico dentro del grado.

Contenido del sitio

  • - About, incluirá una breve descripción sobre ti.
  • - Contact, proporciona un formulario para contactar conmigo.
  • - Degree, proporcionará una descripción detallada de mi grado.
  • - Net, contiene información sobre las paginas de otros compañeros.
  • - Topic, ofrece información completa sobre un tema elegido.

Elementos utilizados

  • - HTML
  • - CSS
  • - Git y GitHub

Estructura del sitio

El sitio web está estructurado en varias elementos interconectadas:

  • - Archivo principal (index.html)
  • - Carpeta CSS para estilos
  • - Carpeta public para páginas adicionales, que antes se han mencionado
  • - Carpeta images para imágenes
