Documento detallado de diseño (DDD)
Descripción de su trabajo:
    Este trabajo consiste en el desarrollo de una página web personal para la asignatura Fundamentos de la Ingeniería Informática del primer curso del grado.

    El objetivo principal es mostrar los conocimientos adquiridos en HTML y CSS, organización y estructura del sitio.
    Se organizo por carpetas: en la raiz se encuantran el archivo .gitignore, LICENSE, README.md y una carpeta llamada docs, dentro de ella esta la carpeta CSS con un solo archivo .css, carpeta images donde se reunen todas las fotos utilizadas, index.html que es la pagina de inicio, y la carpeta public donde se encuantran el resto de paginas del sitio web, concretamente hay 6. 

    La página web está compuesta por varias partes: index (página principal), about (información personal), contact (formulario de contacto), net (enlaces a páginas de otros compañeros), topic (hablo sobre un tema que me interesa "las sakuras") y fii (donde hablo más en profundidad de la asignatura Fundamentos para Ingenieria Informatica).

    Por otra parte, he utilizado un mismo CSS para todas las páginas, utilizando clases para referirme a cosas en particular, como puede ser la galería para estructurar las fotos.

    Para finalizar, el repositorio lo gestione por media de GitHub, utilizando los comandos git init, git add . y git push en la terminal de Visual Studio. Aunque para crear la Licencia utilice git pull para que se subiera a local. 



Problemas durante el desarrollo:
    Al principio tuve problemas con el repositorio, ya que el proyecto se veía de forma diferente en Visual Studio Code que en GitHub. Aunque no supe exactamente cuál fue la causa, el problema se resolvió a lo largo del desarrollo de la web.

    No sabía qué era una licencia, pero no supuso una gran dificultad, ya que busqué información en Internet y pude realizarlo sin problemas. Cree un archivo nuevo en el repositorio gracias a GitHub y déspues hice un git pull para que estubiera dentro de la carpeta raiz. 

    Y tambien tuve problemas con el diseño por los margenes, pero lo acabe solucionando buscando información en W3schools.



Conclusiones:
    La realización de este proyecto ha sido muy útil para adquirir conocimientos básicos de desarrollo web y para aprender a buscar información de forma autónoma cuando surgían dudas o problemas.

    Además, el trabajo me ha permitido conocer mejor la estructura y los contenidos de mi carrera, ya que he tenido que investigar aspectos más específicos como las asignaturas y los resultados de aprendizaje.

    Considero que este proyecto ha sido una buena introducción al desarrollo web y al uso de Git y GitHub.
